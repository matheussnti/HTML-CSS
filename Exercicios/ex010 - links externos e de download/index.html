Usando Links Internos

Links internos são aqueles que apontam para outras páginas ou seções dentro do mesmo site ou documento.

Para acessar a próxima página do site, clique aqui.

Você também pode acessar a página anterior clicando aqui.

Explicação dos Atributos de Link

  • rel="nofollow": Indica aos motores de busca para não seguir o link. É útil quando você não deseja passar autoridade de SEO para o site de destino. Exemplo: Palmeiras.
  • rel="dofollow": É o comportamento padrão dos links. Indica aos motores de busca para seguir o link e passar autoridade. Exemplo: Wikipedia.
