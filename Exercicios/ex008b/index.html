O comenado document.getElementByld('teste') é escrito em linguagem JavaScript

O comando code irá criar uma fonte mono-espassada, dará o mesmo espaço de caracter a todos de forma igual

O comando a seguir, servirá para mostrar o texto exatamente como se encontra no laboratorio:

Sem o comando 'pre'

num = int(input('Digite um numero')) if num % 2 == 0; print('O numero {num} é PAR') else: print('O numero {num} é IMPAR') print('Fim do progama')

Com o comando envelopado 'pre'

num = int(input('Digite um numero'))
if num % 2 == 0;
print('O numero {num} é PAR')
else:
print('O numero {num} é IMPAR')
print('Fim do progama')
