Comandos CSS e Suas Funções

Este documento fornece uma visão geral dos principais comandos CSS e suas funções.

Seletores

Seletores Básicos

element - Seleciona todos os elementos do tipo especificado.

.classe - Seleciona todos os elementos com a classe especificada.

#id - Seleciona o elemento com o ID especificado.

Seletores Combinados

element1, element2 - Seleciona todos os elementos do tipo element1 e element2.

element1 element2 - Seleciona todos os element2 dentro de element1.

Propriedades de Texto

color - Define a cor do texto.

font-size - Define o tamanho da fonte.

font-family - Define a família da fonte.

Propriedades de Layout

display - Define o tipo de caixa de renderização.

position - Define o método de posicionamento de um elemento.

top, left, right, bottom - Define a posição de um elemento posicionado.

Propriedades de Caixa (Box Model)

margin - Define a margem externa de um elemento.

padding - Define a margem interna de um elemento.

border - Define a borda ao redor de um elemento.

Propriedades de Cor e Fundo

background-color - Define a cor de fundo de um elemento.

background-image - Define a imagem de fundo de um elemento.

Propriedades de Bordas

border - Define a borda ao redor de um elemento.

border-radius - Define os cantos arredondados das bordas de um elemento.

Propriedades de Efeitos

box-shadow - Adiciona uma sombra em volta de um elemento.

text-shadow - Adiciona uma sombra ao texto.

Propriedades de Transformação e Animação

transform - Aplica transformações 2D ou 3D a um elemento.

transition - Define as transições entre os estados de uma propriedade.

animation - Define as animações de um elemento.

Propriedades de Flexbox

display: flex - Define um contêiner flexível.

justify-content - Alinha os itens do contêiner ao longo do eixo principal.

align-items - Alinha os itens do contêiner ao longo do eixo transversal.

Propriedades de Grid

display: grid - Define um contêiner de grade.

grid-template-columns - Define a estrutura das colunas na grade.

grid-template-rows - Define a estrutura das linhas na grade.

Outras Propriedades Úteis

visibility - Define a visibilidade de um elemento.

opacity - Define a opacidade de um elemento.

z-index - Define a ordem de empilhamento de elementos sobrepostos.

Conclusão
